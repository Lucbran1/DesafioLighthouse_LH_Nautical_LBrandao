# Desafio Lighthouse – LH Nautical

Autor: Luciano Brandao

## Objetivo
Analisar dados de vendas e clientes da LH Nautical para gerar insights de negócio,
prever demanda e propor sistemas de recomendação.

## Estrutura do Projeto

01_data
- raw: dados brutos fornecidos
- processed: dados tratados

02_notebooks
- análise exploratória e modelagem

03_sql
- queries utilizadas no projeto

04_src
- funções e scripts auxiliares

05_outputs
- gráficos e tabelas geradas

06_docs
- documentação e relatório final

## Etapas do Projeto

1. Exploração dos dados (EDA)
2. Tratamento de dados
3. Análise de vendas
4. Análise de clientes
5. Previsão de demanda
6. Sistema de recomendação
